Qudian enables marking and sharing keywords on web pages. These keywords (namely Qudians) are used to locate information and share interests. Users can rely on these Qudians for seamless reading experience on mobile devices too.

== Description ==

Qudian is similar to social bookmarking services with addition of keyword selection. Qudian helps users to share reading clues by adding and liking Qudians. Users can login using their social networking sites (for now, Sina Weibo is supported). In addition, users build personal interest index over time. Qudian augments reading experience by promoting collaborative reading and enhancing reading efficiency.

== Installation ==

1. Search for qudian in `http://wordpress.org/extend/plugins` and download the plugin as zip file
1. login to `wp-admin`, choose plugin section, upload the zip file, and install the qudian plugin
1. Qudian should be up and running automatically

Alternatively, one can try the following steps.

1. Upload `plugin-name.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Place `<?php do_action('plugin_name_hook'); ?>` in your templates
